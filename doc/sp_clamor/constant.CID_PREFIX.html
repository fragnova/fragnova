The IPFS CID prefix used to use to obtain any data that is stored on the Fragnova Blockchain

The format of the CID prefix is: (see: https://proto.school/anatomy-of-a-cid/05)

0x01 stands for CID v1. 0x55 is the Multicodec code for raw (https://github.com/multiformats/multicodec) 0xa0e402 is the Multihash code for blake2b-256 (https://github.com/multiformats/multihash) 0x20 is the length of the digest
